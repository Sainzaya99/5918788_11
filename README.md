# 다항식 곱셈 프로그램 (과제 11)

## 1. 소개 
이 프로그램은 두 개의 다항식을 곱하는 프로그램입니다. 
배열을 사용해서 다항식을 저장하고 계산했습니다.

## 2. 주요 기능 
*   **입력**: `3X^2+2X^1+1X^0` 처럼 한 번에 입력할 수 있습니다.
*   **계산**: 수학 공식처럼 각 항을 곱하고, 지수를 더해서 결과를 만듭니다.
*   **수정**: 예전 수업 시간에 배웠던 덧셈 코드의 문제를 해결했습니다.

## 3. 사용 방법
1.  프로그램을 실행합니다.
2.  첫 번째 식을 입력합니다 (예: `1X^1+2X^0`).
3.  두 번째 식을 입력합니다 (예: `1X^1+1X^0`).
4.  곱셈 결과가 화면에 나옵니다.

## 4. 예시 
*   입력 1: `1X^1+2X^0`
*   입력 2: `1X^1+1X^0`
*   결과: `1.0X^2+3.0X^1+2.0X^0`
